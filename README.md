# ✈️ Airline Route & Pricing Analytics Dashboard

## 📌 Project Overview

This project analyzes airline ticket pricing trends to identify optimal booking windows and pricing patterns across routes and airlines.

The goal is to provide actionable insights for both customers and businesses to make data-driven decisions.

---

## 🎯 Objectives

* Identify the best time to book flights
* Analyze price variation across routes and airlines
* Understand last-minute pricing trends
* Provide dynamic booking recommendations

---

## 📊 Tools & Technologies

* Power BI (Data Visualization & Dashboard)
* DAX (Dynamic Measures & Insights)
* Excel/CSV (Data Source)

---

## 📈 Key Insights

* Prices increase significantly in the last 7–10 days before departure
* Early booking results in lower ticket prices
* Certain routes consistently have higher pricing
* Airlines follow different pricing strategies (premium vs competitive)

---

## 🚀 Dashboard Features

* Interactive filters (Airline, Route, Class)
* Dynamic insights using DAX
* Booking recommendation logic
* Visual analysis of pricing trends

---

## 📂 Project Structure

* Dataset: Raw airline data
* PowerBI: Dashboard file (.pbix)
* Screenshots: Dashboard previews

---

## 🧠 Business Impact

This dashboard helps:

* Customers choose the best time to book flights
* Airlines optimize pricing strategies
* Travel platforms improve recommendations
